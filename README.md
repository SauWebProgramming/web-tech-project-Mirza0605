Mirnet – Film & Dizi Platformu

Mirnet, kullanıcıların film ve dizileri arayabildiği, detaylarını görüntüleyebildiği ve kendi listelerine/favorilerine ekleyebildiği modern bir web uygulamasıdır.
TMDB API kullanılarak geliştirilen bu proje tamamen SPA (Single Page Application) mantığıyla çalışır.

Özellikler

- Film & Dizi Arama

- İsim, tür ve yıla göre filtreleme

- TMDB’den gerçek zamanlı veri çekme

- Modern arayüz ve responsive tasarım

- Detay Sayfası

- Poster, yıl, tür, puan, oyuncular ve özet

- Favorilere ekle / kaldır

- Listeme ekle / kaldır

- Dinamik buton durumu (Favorilerde ✓, Listemde ✓)

- Favori Sistem

- LocalStorage kullanır

- Kullanıcı favoriye eklediyse buton otomatik güncellenir

- Favoriden kaldırma özelliği vardır

- Kişisel Liste Sistemi

- Eklenen öğeler "Listem" sayfasında gösterilir

- Listeden çıkarılabilir

- Tasarım favorilerle birebir uyumludur

- Tamamen SPA Yapısı

- Sayfa yenilenmeden içerik yüklenir

- Router mantığı hash (#) tabanlıdır

- Tüm sayfalar JS ile dinamik oluşturulur
